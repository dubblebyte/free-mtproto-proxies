Manual Setup

If the Connect buttons don't work automatically:

  1. Open Telegram → Settings → Data and Storage → Proxy
  2. Tap Add Proxy → Select MTProto
  3. Enter the server, port, and secret from proxies.json
  4. Tap Save

Frequently Asked Questions

Is this free to use?

Yes. 100% free. No accounts, no bandwidth limits, no premium tier.

Is it safe to use these proxies?

For Telegram traffic, yes. MTProto proxies only relay Telegram traffic and cannot read your messages (Secret Chats are end-to-end encrypted). The proxy operator can see your IP address and connection time, but nothing inside the traffic. Do not route sensitive non-Telegram traffic through public proxies.

Why update every 8 hours?

Public proxies are ephemeral. Servers go offline or get blocked without warning. An 8-hour cycle keeps the list fresh without being excessive.

Can I use this data in my own app or bot?

Absolutely. proxies.json is machine-readable and updated automatically. No API key, no rate limit, no attribution required.
